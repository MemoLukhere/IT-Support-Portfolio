# Ticketing System Lab

## Overview

This project demonstrates my hands-on practice with IT help desk operations and ticket lifecycle management using Spiceworks. I created and managed simulated support tickets based on realistic workplace scenarios to strengthen my troubleshooting, documentation, and incident management skills.

## Objectives

* Understand the complete ticket lifecycle from creation to closure
* Practice categorizing and prioritizing support requests
* Document troubleshooting steps and resolutions
* Develop familiarity with common IT support workflows
* Improve technical documentation and problem-solving skills

## Tools Used

* Spiceworks Help Desk
* Windows Operating System
* Command Prompt (CMD)

## Skills Demonstrated

* Incident Management
* Ticket Lifecycle Management
* Technical Documentation
* Priority Assignment
* Network Troubleshooting
* User Account Support
* Hardware and Printer Support
* Windows Troubleshooting
* Customer-Focused Problem Solving

## Ticket Scenarios Completed

| Ticket                       | Category           | Status |
| ---------------------------- | ------------------ | ------ |
| User Cannot Connect to Wi-Fi | Network            | Closed |
| Password Reset Request       | User Account       | Closed |
| Windows Updates Failing      | Software           | Closed |
| Printer Offline              | Hardware / Printer | Closed |
| Network Adapter Disabled     | Network            | Closed |

##  Workflow

For each ticket, I followed a structured process:

1. Created the support ticket.
2. Assigned an appropriate category and priority.
3. Investigated the reported issue.
4. Performed troubleshooting steps.
5. Documented the resolution.
6. Closed the ticket after confirming the issue was resolved.

## Key Takeaways

Completing this lab improved my understanding of how IT support teams manage incidents, communicate resolutions, and maintain accurate documentation. It also reinforced practical troubleshooting techniques for common issues encountered in enterprise environments.
