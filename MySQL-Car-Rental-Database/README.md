# Car Rental Database Management System

## Overview

This project is a MySQL-based relational database designed for a fictional multi-branch car rental company. It manages branches, employees, vehicles, customers, rentals, payments, and maintenance records while demonstrating sound database design principles and SQL querying techniques.

## Why I Built This

As a Computer Network Engineering graduate interested in IT support and systems administration, I wanted to strengthen my database skills by designing and implementing a practical business application. This project demonstrates my ability to create relational databases, enforce data integrity using primary and foreign keys, and write SQL queries that support day-to-day business operations.

## Database Structure

The database consists of eight interconnected tables:

* Branches
* Employees
* Vehicle Categories
* Vehicles
* Customers
* Rentals
* Payments
* Maintenance

Relationships are enforced using foreign keys. For example:

* Each employee belongs to a specific branch.
* Each rental links a customer, vehicle, and employee.
* Each payment is associated with a rental transaction.
* Maintenance records are linked to individual vehicles.

## Features Implemented

* Relational database schema using MySQL
* Primary key and foreign key constraints
* Sample data representing multiple branches and rental transactions
* Vehicle availability tracking
* Revenue reporting by branch
* Maintenance cost reporting
* Multi-table JOIN queries
* Aggregate functions (`SUM`)
* Filtering and sorting using `WHERE` and `ORDER BY`
* A database view (`active_rentals_view`) for monitoring active and overdue rentals

## Example Business Queries

Examples of analyses performed include:

* Display available vehicles for rent
* List customers alphabetically
* Identify vehicle categories below a specified daily rate
* Display rentals with customer and vehicle information
* Show which employee processed each rental
* Calculate total revenue generated by each branch
* Calculate maintenance costs per vehicle
